📖 What is CompSci Leveling?

CompSci Leveling is a gamified learning website where you master HTML and JavaScript — the two core building blocks of every website — by completing ranked modules, exactly like leveling up in an RPG. Forget passive reading. Here, you earn XP, unlock harder modules, trigger random Gate events, and climb the rank ladder from F all the way to the legendary S Rank.

📚

Study First

Every module opens with a reviewer — a full lesson with explanations, code examples, and tips. Read it, understand it, then take the quiz.

❓

Quiz to Earn XP

Answer 5 questions per module. Each correct answer earns XP. Higher rank modules reward more XP. Stack it up to rank up.

⚡

Gate Events

A mysterious Gate may randomly appear when you enter a module. Accept the challenge and earn +50 bonus XP. Decline and move on.

🧪

Live Sandbox

Practice what you learn in the built-in code editor — HTML, CSS, and JS panes with a live preview. Write. Break. Fix. Repeat.

🏆 The Rank System

Complete modules, earn XP, and rank up. Each rank unlocks harder content and bigger rewards. Can you reach S Rank?
